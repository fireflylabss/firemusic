# FireMusic Roadmap

## Overview

FireMusic is a music player built for developers and power users who want control over their music library and listening experience.

**Status**: v0.1.0 (MVP)

---

## v0.2.0 - "Library Management" (Jun 2026)

- [ ] Auto-organization of library
- [ ] Smart playlists (automatic rules)
- [ ] Metadata editing (tags, cover art)
- [ ] Duplicate detection
- [ ] Library cleanup
- [ ] Import from iTunes/Spotify playlists
- [ ] Volume normalization
- [ ] Gapless playback

---

## v0.3.0 - "Discovery" (Aug 2026)

- [ ] Last.fm scrobbling
- [ ] Music recommendations
- [ ] Similar artists
- [ ] Genre exploration
- [ ] New releases
- [ ] Top charts
- [ ] Radio mode (based on preferences)

---

## v0.4.0 - "Streaming" (Oct 2026)

- [ ] Spotify integration
- [ ] YouTube Music integration
- [ ] SoundCloud integration
- [ ] Bandcamp integration
- [ ] Local + streaming unified
- [ ] Offline cache for streaming

---

## v0.5.0 - "Social" (Dec 2026)

- [ ] Share playlists
- [ ] Collaborative playlists
- [ ] Friends activity
- [ ] Music social network
- [ ] Comments on tracks
- [ ] Follow artists

---

## v1.0.0 - "Platform" (Feb 2027)

- [ ] Mobile app
- [ ] Desktop app with mini player
- [ ] Car integration (Android Auto, CarPlay)
- [ ] Smart speakers (Alexa, Google Home)
- [ ] Hi-Res audio support
- [ ] Customizable EQ

---

## Integration with Fire Suite

- **firekeep**: Credentials for streaming services
- **firetasks**: Focus mode tasks while listening
- **fireworker**: Mini player widget in hub
- **firenotes**: Playlist notes and descriptions
